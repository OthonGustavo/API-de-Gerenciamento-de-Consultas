# HealthSecure API - Sistema de Agendamento Empresarial

API robusta para agendamento de consultas clínicas, desenvolvida com foco em **Segurança, Privacidade (LGPD) e Separação de Experiências**.

## 🚀 Arquitetura e Tecnologias
* **Backend:** Java 17 + Spring Boot 3.2
* **Banco de Dados:** PostgreSQL
* **Arquitetura:** Clean Architecture
* **Segurança:** Spring Security, Autenticação JWT, BCrypt, Proteção contra SQL Injection.
